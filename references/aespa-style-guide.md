# aespa Style Guide for Error Workbooks

## Document Formatting

### Page Setup
- Font: 仿宋 (body), 黑体 (headings)
- Size: 小四号 (12pt)
- Line spacing: 1.15
- Paragraph spacing: minimum (3pt)
- Space between questions: 3 blank lines

### Title Format
```
九下 第X周错题集 aespa Edition
整理日期：YYYY年MM月DD日
```

## Member Configuration

### Karina (柳智敏) 💙 - The Gentle Leader
- **Role**: Leader, reassuring presence
- **Tone**: Encouraging, guiding, supportive
- **Quote style**: " mistakes help us grow", "you've got this"
- **Color**: Blue (0, 102, 204)
- **Usage**: Opening encouragement, difficult concept explanations

### Giselle (内永绘里) 🖤 - The Cool Rapper  
- **Role**: Cool confidence, swagger
- **Tone**: Confident, slightly edgy, direct
- **Quote style**: "easy peasy", "crush this"
- **Color**: Purple (102, 0, 153)
- **Usage**: After medium-difficulty questions, hype moments

### Winter (金玟庭) ❄️ - The Cute One
- **Role**: Cheerful, warm encouragement
- **Tone**: Sweet, adorable, uplifting
- **Quote style**: "you're doing great!", "fighting!"
- **Color**: Light Blue (0, 153, 204)
- **Usage**: After mistakes, encouragement sections

### Ningning (宁艺卓) 🦋 - The Straightforward Maknae
- **Role**: Honest, relatable, no-filter
- **Tone**: Direct, slightly sassy, down-to-earth
- **Quote style**: "that was tricky", "don't worry about it"
- **Color**: Orange-Red (204, 51, 0)
- **Usage**: Common mistakes, relatable struggles

## Quote Placement Rules

### Part 1: Original Questions (练习用)

**After each major question:**
- Difficult/conceptual → Karina 💙
- Standard difficulty → Giselle 🖤
- Common mistakes → Ningning 🦋
- Encouragement needed → Winter ❄️

**Section ending** (all four members):
```
💙 Karina: "Keep going, you're doing amazing!"
🖤 Giselle: "Next level incoming!"
❄️ Winter: "Fighting! We believe in you!"
🦋 Ningning: "Almost there, don't give up!"
```

### Part 2: Answers & Explanations (参考用)

**Opening** (Karina only):
```
💙 Karina: "Welcome back! Let's review together~"
```

**After each explanation:**
- Based on member who commented in Part 1
- Or rotate through members systematically

## Color Application

### Member Quotes
- Text color: Member's assigned RGB
- Can be applied to full quote or just member name/emoji

### Key Elements
- Correct answers: Bold + Blue highlight (0, 102, 204)
- Student wrong answers: Normal text
- Explanations: 仿宋, compact spacing

## Image Handling

### Source Images

- Width: 5.5 inches
- Caption: `📷 【物理XX 原卷图示】`
- Position: At start of each subject section
